1. 파티션 작업

  • DiskUtility에서 필요한 용량만큼 파티션을 나눈다
  • asahi-fedora OS 설치 용량은 42.5GB로 이보다 큰 용량만큼 할당하여야 한다
 
 

2. 페도라

1. 페도라 설치

  • 아래의 명령어를 실행한다
      • curl https://leifliddy.com/fedora.sh | sh
  • 이후 재부팅을 진행하는데, 전원 버튼을 꾹 눌려서 시동옵션으로 부팅하여야 한다
  • 이후 자동적으로 터미널 화면으로 진입하며, 계정명, 아이디와 비밀번호을 정확하게 입력한 뒤 재부팅을 한다
  • 참고 :
 

2. 페도라 기본 설정

  • 최초 로그인 시 root/fedora로 접속한다
  • 네트워크 연결 (참고)
      • nmcli dev wifi list
sudo nmcli dev wifi connect ${SSID} password "${PASSWORD}"
ping 8.8.8.8
nmcli con show
  • 계정추가 (참고)
      • useradd ${계정명}
passwd ${계정명}
  • GNOME GUI 설치 (참고)
      • dnf -y group install "Basic Desktop" GNOME
startx
  • https://re-atmosphere.tistory.com/89
 

3. ros 설치

  • http://wiki.ros.org/Installation/Fedora
  • http://wiki.ros.org/Installation/Source
 
 

2. Asahi Linux

1. 아사히 리눅스 설치

curl https://alx.sh | sh
 

2. 기본 설정

  • go 언어 설치 (참고)
      • sudo pacman -Sy go
  • yay 설치 (참고)
      • git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
 
 

3. 리눅스 삭제

sudo curl -L https://alx.sh/wipe-linux | sh
